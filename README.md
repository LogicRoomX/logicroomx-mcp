# LogicRoomX MCP Plugins

Real-time data plugins for AI assistants. Built by LogicRoomX.

## Available Plugins

| Plugin | Description | Endpoint |
|--------|-------------|----------|
| [Crypto](./crypto/README.md) | Real-time crypto spread, funding rates, cash & carry | `https://api.logicroomx.com/mcp` |

## What is MCP?

Model Context Protocol (MCP) allows AI assistants to call external tools for real-time data they cannot access on their own.
